# Media License Policy

Contributor default: audio assets referenced by this project should use **Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)** unless otherwise noted in `ATTRIBUTION.md`.

The two current live test tracks in `channels.json` are exceptions. Their Internet Archive item metadata does not currently publish a reuse license field, so they must be treated according to the rights notes recorded in `ATTRIBUTION.md`.

## Requirements

1. Provide attribution to the original creator.
2. If a track is marked `CC BY-SA 4.0`, share derivatives under the same license.
3. If a track has a custom rights note in `ATTRIBUTION.md`, follow that note instead of the default policy.
4. Keep attribution metadata intact when sharing mirrors or derivatives.

## Practical Attribution Format

Use one of these formats when reusing media:

`Track Title - Artist Name (CC BY-SA 4.0) Source: https://...`

`Track Title - Artist Name (See ATTRIBUTION.md rights note) Source: https://...`

## Scope

- This media policy applies to channel audio and other media listed in `ATTRIBUTION.md`.
